Libraries einbetten
-------------------

Wie *ziehe* ich die Libraries *an*?

* Java: Maven (`pom.xml`)
* .Net: nuget (`nuget.config`)
* Python: pip (`requirements.txt`)
* Rust: Cargo (`config.toml`)
* C/C++: bitbake, apt/yum, conan, cmake, header-only

### Design

* je nach Level des Projektes
* nicht manuell
